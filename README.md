# Microsoft Entra Identity and Access Operations

This project documents hands-on lab work focused on Microsoft Entra identity administration, tenant operations, authentication, and access control.

The goal of this project is to build practical, job-relevant proof of work for IT support, junior cloud support, and hybrid Microsoft administration roles.

## Why this project is split into two parts

To keep the project clean and reviewer-friendly, I separated it into:

- **[01-Core-Identity-and-Tenant-Operations/]**(https://github.com/richardbvu/microsoft-entra-identity-and-access-operations/tree/main/01-Core-Identity-and-Tenant-Operations) focuses on core tenant and identity tasks
- **[02-Identity-Control-and-Access-Operations/]**(https://github.com/richardbvu/microsoft-entra-identity-and-access-operations/tree/main/02-Identity-Control-and-Access-Operations) builds on that foundation by revisiting licensing-blocked topics properly and adding deeper policy-driven access controls

That makes the project more honest, more organized, and more representative of how identity work actually grows in real environments.
