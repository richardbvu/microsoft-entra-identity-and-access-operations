# Microsoft Entra Identity And Access Operations Lab

## Overview
This project documents the build, validation, and troubleshooting of a Microsoft Entra (Azure AD) lab environment.

The lab includes:
- Cloud-only user identity management
- Security group configuration and membership validation
- Administrative Unit (AU) structure and scoped administration concepts
- Authentication and sign-in behavior testing
- Identity-based troubleshooting scenarios

This project was documented using an evidence-first workflow, with build evidence, troubleshooting evidence, ticket write-ups, and resume/interview packaging aligned to real IT support practices.

## Skills Demonstrated
- Cloud identity lifecycle management in Microsoft Entra
- User authentication and sign-in troubleshooting
- User Principal Name (UPN) validation and identity format awareness
- Account state troubleshooting (enabled vs disabled accounts)
- Security group creation and membership management
- Understanding of assigned vs dynamic membership (with licensing limitations)
- Role-based access control (RBAC) concepts and limitations in Entra ID Free
- Scoped administration using Administrative Units
- Differentiation between identity management and access management
- Structured troubleshooting methodology and documentation

## Project Structure
